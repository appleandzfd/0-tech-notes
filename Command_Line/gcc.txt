-I option set the include path with self-defined path.
e.g: 
   gcc myapp.c -I /home/liuchang/include -o myapp

-L option set the libraray with self-defined path(not standard path)
e.g:
   gcc myapp.c -L/home/liuchang/lib -I/home/liuchang/inlucde -o myapp

-o option set the output file name
(without -o then the file name is a.out)
e.g:
   gcc myapp.c -o myName
